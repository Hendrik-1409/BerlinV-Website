Allgemeine Hinweise

Die folgenden Hinweise geben einen einfachen Überblick darüber, was mit Ihren personenbezogenen Daten passiert, wenn sie auf den FiveM Server BerlinV sich verbinden und mit diesem interagieren. Personenbezogene Daten sind alle Daten, mit denen Sie persönlich identifiziert werden können. Ausführliche Informationen zum Thema Datenschutz entnehmen Sie unserer unter diesem Text aufgeführten Datenschutzerklärung.

Datenschutz

Die Betreiber des FiveM Servers BerlinV nehmen den Schutz Ihrer persönlichen Daten sehr ernst. Wir behandeln Ihre personenbezogenen Daten vertraulich und entsprechend der gesetzlichen Datenschutzvorschriften sowie dieser Datenschutzerklärung.
Wenn Sie mit dem FiveM Server BerlinV interagieren (verbinden, spielen etc.), werden verschiedene personenbezogene Daten erhoben. Personenbezogene Daten sind Daten, mit denen Sie persönlich identifiziert werden können. Die vorliegende Datenschutzerklärung erläutert, welche Daten wir erheben und wofür wir sie nutzen. Sie erläutert auch, wie und zu welchem Zweck das geschieht.
Wir weisen darauf hin, dass die Datenübertragung im Internet (z.B. bei der Kommunikation per E-Mail) Sicherheitslücken aufweisen kann. Ein lückenloser Schutz der Daten vor dem Zugriff durch Dritte ist nicht möglich.

Logs

Der FiveM Server BerlinV benutzt den Server logger “JD-logs”. Dieser ist kostenfrei für jeden Server auf www.github.com verfügbar. Auf dem Discord Server BerlinV werden folgende Daten durch “JD-logs” dem Team (erkenntlich an der Discord- Rolle “Team”) freigegeben:

  • Die Nachricht/ der Command, die/ der in den Chat des Servers eingegeben wurde
  • Der FiveM/Steam Name
  • Die Player ID, die der Server dem Spieler beim connecten auf den FiveM Server BerlinV zugewiesen hat (dient zur Identifikation der Spieler auf dem Server)
  • Der Discord Name, inkl. der Discord ID
  • Die Steam ID, mit entsprechender weiterleitung zur Seite www.steamcommunity.com, wodurch man das Steam Profil des Spielers einsehen kann.
  • Die einmalige FiveM Lizenz, die dem Spieler durch FiveM zugewiesen wurde
  • Die IP des jeweiligen Spielers ist nur für Personen einsehbar, die die Rolle stlv. Leitung oder Leitung einnehmen!

Wir erheben diese Daten, zur Sicherheit des Servers und der Spieler. Diese Daten werden nicht an dritte weitergegeben, niemand außer Personen, die dem Team angehören können diese Daten einsehen. Der Server ist mit den Endgerät des Spielers Ende-zu-Ende verschlüsselt
